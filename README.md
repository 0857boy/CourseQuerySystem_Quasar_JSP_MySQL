# 課程查詢系統

## 架構
![心智圖](./Imgs/Mind.png)

- 前端: Quasar
- 後端: JSP(tomcat server)
- 資料庫: MySQL (or sqlite oracle)

## 系統功能
- 查詢課程
- 新增課程
- 刪除課程
- 修改課程

## Poxy
- 設定代理伺服器，解決跨域問題(前端與後端不同埠號)。
